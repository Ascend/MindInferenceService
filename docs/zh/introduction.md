# 简介<a name="ZH-CN_TOPIC_0000002496450893"></a>

昇腾推理微服务MIS（Mind Inference Service）提供了大模型支持，无需复杂的依赖安装，即可快速完成部署。针对昇腾硬件进行深度的性能优化，省去繁琐的调优过程。提供行业标准接口，便于集成到企业业务系统中，助力业务高效运行。

**使用引导<a name="section72154313388"></a>**

如果第一次使用本软件，请根据[安装部署](installation_guide.md#安装部署)准备好相关环境和软件包并执行安装与部署。 如果对相关流程比较熟悉，可以直接通过[支持的接口列表及参数](api.md#支持的接口列表及参数)获取需要的接口。
